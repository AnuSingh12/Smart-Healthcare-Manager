#Smart Healthcare Manager (Desktop Application)

A desktop-based healthcare management system built using **Java and MySQL**.
This application helps manage hospital operations such as patient records, doctor details, appointments, and billing.

#Features

*  Doctor Management
*  Patient Management
*  Appointment Scheduling System
*  Billing System with PDF Generation
*  Database Integration using MySQL

#Tech Stack

* Java (Core Java / JavaFX)
* MySQL Database
* JDBC (Java Database Connectivity)
* PDF Generation

#Installation

1. Clone the repository:
   git clone https://github.com/your-username/Healthcare-Manager.git

2. Open project in your IDE (IntelliJ / Eclipse)

3. Setup MySQL:

   * Create database: `hospital_management`
   * Import required tables

4. Update DB credentials in code

5. Run the application
